# CyanoSafe

A web application for reviewing harmful algal bloom (HAB) event monitoring data in California.

## Overview

CyanoSafe helps users detect, track, and respond to cyanobacterial (blue-green algae) blooms that can pose risks to human health, wildlife, and water quality across California's lakes, reservoirs, and waterways. The promary data used in this app is from the Freshwater and Estuarine HAB (FHAB) program run at the CA State Water Resources Control Board. This data - https://data.ca.gov/dataset/surface-water-freshwater-harmful-algal-blooms - has been collected for over 12 years and has some unique characteristics that are impoirtant to consider in a risk communication produt like this web app.

Access the web app demo here: https://ggearheart.github.io/CyanoSafe_demo/

## IMPORTANT CONSIDERATIONS

Each of the events on this map reflect what was discovered after an alleged HAB event. The process usually begins with a phone call or email to the program of a suspected HAB event. Once received by staff, someone is sent to investigate the alleged event. An event can be confirmed in a number of different ways. A small percentage of the event investigations involve water sampling and mutiple types of laboratory analysis - microscopy, genetic testing for toxin producing cyanobacteria, and chemical analysos for cyantoxins present in the sample. So to summarize, this data reflects reactive data collection methods (invesitgation response collected data, not ambient or study design-based collected data) and only looks for toxins present in some of the investigations. This means toxins are likely to be present in many more HAB events in this dataset than the "toxin confirmed" filter suggests.

## Features

- Identify potential harmful algal bloom events
- Monitor water bodies across California
- Provide timely alerts and risk assessments
